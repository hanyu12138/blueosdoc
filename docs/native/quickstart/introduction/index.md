# 概述

除了 JS API，我们还针对高性能的需求场景提供 Native API。当前，我们开放了部分能力的 Native API，并支持 C、C++标准库及标准 POSIX 接口。

现阶段，以定向合作的形式向合作方提供 Native 开发能力，合作方通过调用 Native API，作为系统能力的一部分提供给 JS 侧开发使用。 Native 能力将在后续版本逐步扩展。

# barcode

条形码，将文本内容转换为条形码展示。

### 子组件

不支持

### 属性

支持[通用属性](../../common/common-attributes/index.md)

| 名称 | 类型 | 默认值 | 必填 | 描述 |
| --- | --- | --- | --- | --- |
| value | `<string>` | - | 是 | 条形码内容，码制为 Code128 码，长度小于等于 20 字节 |

支持[通用样式](../../common/common-styles/index.md)

支持[通用事件](../../common/common-events/index.md)

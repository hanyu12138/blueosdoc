# stack

基本容器，子组件排列方式为层叠排列，每个直接子组件按照先后顺序依次堆叠，覆盖前一个子组件，支持子组件，支持[通用属性](../../common/common-attributes/index.md)

### 样式

支持[<div> 样式](../div/index.md)，支持[通用样式](../../common/common-styles/index.md)

### 事件

支持[通用事件](../../common/common-events/index.md)

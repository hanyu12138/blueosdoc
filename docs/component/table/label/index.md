# label

为 [input](../input/index.md) 组件定义标注

### 子组件

不支持

### 属性

支持[通用属性](../../common/common-attributes/index.md)

| 名称 | 类型 | 默认值 | 必填 | 描述 |
| --- | --- | --- | --- | --- |
| target | `<string>` | - | 否 | 目标 input 组件 id |

### 样式

支持[通用样式](../../common/common-styles/index.md)

支持[text 样式](../../basic/text/index.md)

### 事件

点击 label 组件，触发 target 绑定 id 的 checkbox 或者 radio 组件点击事件，扩大点击范围

# 通用错误码

## 提供公共的错误码

### 其中，错误码 200 为系统通用错误码，所有系统未知异常发生时抛出。比如系统申请内存空间失败等。

| code | 含义 |
| --- | --- |
| 200 | 通用错误。 |
| 202 | 参数错误。 |
| 300 | I/O 错误。 |
